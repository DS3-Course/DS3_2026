# HTP mass cytometry data visualization

1. **Ensure you have the course materials** 
      1. **Clone** this repository (first time only)
         ```bash
         git clone https://github.com/DS3-Course/DS3_2026.git
         ```
      2. **Pull** updates each day to ensure updated content:
         ```bash
         git pull
         ```
         *Alternative:* direct download from GitHub

---

2. **Open the project in RStudio**  
      - **Option A:** Navigate to the folder and double-click the `.Rproj` file  
      - **Option B:** In RStudio, go to `File > Open Project...` and select the `.Rproj` file  

> 💡 Always open the `.Rproj` file first (not individual scripts)
  
> ⚠️ This project uses `renv` for package management (install with `install.packages("renv")` if needed)

---

3. **Run the analysis**

   - Open the R script(s)  
   - Work through the analysis steps in the script  

---

### You can use Git LFS (large file storage) to clone the data files for this exercise from Github.  
ALTERNATIVE - manually download repository as zip file, unzip, and open via Rproj file.  
For details, see [here](https://git-lfs.com/)  

a. Mac OS:  
   - Using [Homebrew](https://brew.sh/): `brew install git-lfs`   
   - Using [MacPorts](https://www.macports.org/): `port install git-lfs`   

b. Windows:  
   - Git LFS is included in the distribution of Git for Windows  

c. Linux:  
   - Instructions [here](https://github.com/git-lfs/git-lfs/blob/main/INSTALLING.md)  

Once installed, set up Git LFS for your user account by running:
`git lfs install`
